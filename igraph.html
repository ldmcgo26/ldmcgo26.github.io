Voronoi Game on Graphs

I created a testing framework for the Voronoi Game, which is a game based on the facility location problem in Algorithmic Game Theory. In essence, two players compete for control over a continuous space by alternating placing facilities. My focus was examining this game on graphs, with more than two players, and with differing value assessments. In order to do so, I created a testing framework in Python to provide data for analysis of various strategies. I also utilized iGraph, which is a python graph library, to visualize the game.
